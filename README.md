# Dixit AI Online

Gra karciana Dixit zintegrowana z AI. Projekt składa się z serwera (Node.js/Prisma) oraz klienta (React/Vite).

## 🚀 Szybki start dla zespołu (Lambdy_PV)

Aby uruchomić projekt, wykonaj poniższe kroki:

1. **Konfiguracja środowiska:**
   Plik `.env` zawiera wrażliwe dane i nie jest przechowywany w repozytorium. 
   - Pobierz plik `.env` z naszego wspólnego dysku **Lambdy_PV**.
   - Umieść go w folderze `/server`.

2. **Pierwsza instalacja:**
   Uruchom skrypt instalacyjny (tylko za pierwszym razem):
   ```bash
   chmod +x install.sh
   ./install.sh
